## 4. H5 用户操作教程

1. 打开 H5：首页 `https://rd.chaimen666.com/?tenantCode=qiwai-showcase#/`。
2. 登录：进入“我的”或报名时跳转登录，选择“密码登录”，输入演示手机号和密码。
3. 浏览活动：点击“活动”或首页精选活动。
4. 免费活动报名：进入免费活动详情，点击“立即报名”，填写报名信息后提交。
5. 付费活动报名：进入 `【演示】家庭教育沟通工作坊`，点击“立即报名”，选择线下收款并提交。
6. 查看报名：进入“我的”查看报名详情，未确认收款前显示待付款或等待确认。
7. 财务确认后刷新：报名详情会变为报名成功，并展示签到码。
8. 签到后刷新：签到核销完成后，报名详情显示已签到。

## 5. 后台操作教程

活动运营：

1. 登录后台：`showcase_admin` 或 `showcase_ops`。
2. 进入“活动”，可查看 6 个演示活动，覆盖免费和收费活动。
3. 进入“报名”，可按手机号、活动、状态筛选报名记录。
4. 进入“营销弹窗”，点击“生效检测”验证 H5 首页弹窗。
5. 进入“首页装修”，可查看当前演示首页，并保留“应用上线简洁版模板”入口。

财务确认线下收款：

1. 登录后台：`showcase_finance`。
2. 进入“订单”。
3. 搜索 H5 用户手机号或订单号。
4. 对待付款订单点击“收款/确认收款”。
5. 填写备注并确认。
6. 回到 H5 报名详情刷新，状态应变为报名成功。

签到核销：

1. 登录后台：`showcase_checkin`。
2. 进入“签到核销”。
3. 输入 H5 报名详情展示的签到码。
4. 可填写现场备注。
5. 点击“核销签到”。
6. H5 报名详情刷新后显示已签到。

商城与店铺：

1. `showcase_store_owner` 可进入“商城商品”“商城订单”，不可进入“代理结算”。
2. `showcase_store_finance` 可进入“商城订单”“商城结算”，不可进入“商城商品”。
3. `showcase_agent_owner` 可进入“代理结算”“商城订单”，不可进入“商城商品”。

系统设置：

1. 使用 `admin` 登录后台。
2. 进入“系统设置 / 部署配置”。
3. 查看 API/Admin/H5 三端版本。
4. 短信配置在同页维护，真实短信验收需要填写腾讯云短信参数。
