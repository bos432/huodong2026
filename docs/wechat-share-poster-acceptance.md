# 慢π微信分享与海报真机验收清单

本文用于补足桌面浏览器无法完整替代的真机验收：微信内打开 H5、生成活动/心得海报、长按保存图片、朋友圈分享、二维码扫码回流、用户图片上传。

## 1. 验收前置条件

- 使用预发或生产 HTTPS 域名，不使用 `localhost`、`127.0.0.1` 或自签证书。
- 后台“系统设置 / 部署配置”已填写真实 H5、后台、API 域名。
- 后台“系统设置 / 部署配置 / 微信 H5 真机验收”已填写验收租户码，并复制真机验收入口或验收记录模板。
- H5 域名、API 域名、上传域名已加入微信相关合法域名或业务域名配置。
- 手机微信可访问 H5 首页、活动详情、共修动态详情、我的心得。
- 至少准备一个已发布活动、一个已签到或已付款报名、一个已审核通过的参与者心得。
- 准备 iOS 和 Android 各 1 台手机；若只能先测一台，必须在验收记录里写明未覆盖机型。

## 1.1 后台验收助手

后台提供一个不直接改服务器的验收助手：

```text
系统设置 -> 部署配置 -> 微信 H5 真机验收
```

该区域可以：

- 根据当前 H5 域名和租户码生成首页、活动列表、共修动态、发布心得、我的心得、今日打卡等验收入口。
- 复制一份真机验收记录模板，便于运营在手机微信测试时逐项填写。
- 记录验收状态、验收时间和备注，随部署配置一起保存。
- 提示当前 H5/API 域名是否仍是本地域名、示例域名或非 HTTPS。

该区域不能替代手机微信测试。长按保存海报、朋友圈卡片、扫码回流、微信缓存和图片上传权限仍必须在真实 iOS/Android 微信中完成。

## 3. 活动详情分享链路

1. 在微信内打开 H5 活动详情。
2. 确认页面展示活动标题、封面、时间、地点、费用、报名按钮和底部导航。
3. 点击页面或浏览器右上角分享入口，将链接发送给另一个微信账号或群聊。
4. 使用接收账号打开分享链接。
5. 确认打开的是同一个活动详情，未登录用户也能浏览公开内容。
6. 如链接携带邀请码或分享参数，后台活动漏斗应能看到分享访问增长。

通过标准：

- 微信内链接可打开，不出现白屏、证书错误、跨域错误或路由丢失。
- 分享卡片标题、描述、缩略图符合慢π品牌和活动内容。
- 分享回流后 `tenantCode`、活动 id 和页面路由不丢失。

## 4. 参与者心得发布与公开展示

1. 使用已参加活动的用户登录 H5。
2. 从活动详情、报名详情或“我的心得”进入发布心得。
3. 选择“我的已参加活动”。
4. 输入 20 字以上心得正文。
5. 上传 1 张图片，再补测上传 9 张图片。
6. 提交后确认状态为待审核。
7. 后台共修动态管理中筛选“参与者心得 / 待审核”，审核通过。
8. 回到微信 H5 首页、共修首页、活动详情口碑区，确认心得公开展示。
9. 刷新页面后再次确认图片、正文、作者、关联活动仍正常。

通过标准：

- 未参加活动用户不能发布关联心得，并看到明确提示。
- 图片选择、压缩或上传失败时有清晰提示。
- 审核前不公开，审核通过后公开，驳回或下架后前台不可见。

## 5. 心得海报生成与长按保存

1. 在微信内打开已审核通过的心得详情页。
2. 点击“生成海报”。
3. 等待海报生成完成。
4. 检查海报内容：
   - 慢π品牌标识。
   - 活动标题。
   - 心得摘要。
   - 用户昵称。
   - 首图或活动封面。
   - 二维码或可复制分享链接。
5. 长按海报，选择保存到相册。
6. 在系统相册打开保存的图片，确认图片完整、无黑块、无跨域污染导致的空白。
7. 使用另一台手机扫描海报二维码。
8. 确认扫码后进入同一条心得详情，未登录也可查看公开内容。

通过标准：

- 海报生成不白屏、不缺图、不遮挡文字。
- 长按保存成功，保存后图片清晰。
- 二维码或分享链接可回流到正确心得详情。
- iOS 和 Android 均通过；如有差异，必须记录机型、微信版本和截图。

## 6. 朋友圈传播验收

1. 将活动详情链接分享到朋友圈，检查标题、描述、缩略图。
2. 将心得详情链接分享到朋友圈，检查标题、描述、缩略图。
3. 将保存的心得海报作为图片发布朋友圈。
4. 使用另一个微信账号从朋友圈打开链接或扫码海报。

通过标准：

- 文案不出现“公开募捐”“捐款认领”“募捐目标”等高风险表述。
- 朋友圈卡片能正确打开 H5。
- 海报图片在朋友圈缩略图中可读，二维码区域没有被压缩到不可识别。

## 7. 常见问题排查

| 问题 | 优先检查 |
| --- | --- |
| 微信内白屏 | HTTPS 证书、H5 构建产物、API 反向代理、路由 base、浏览器控制台错误 |
| 图片上传失败 | upload 合法域名、文件大小、MIME 类型、上传目录权限、`PUBLIC_API_ORIGIN` |
| 海报首图空白 | 图片跨域头、图片协议是否 HTTPS、canvas 是否被跨域污染 |
| 二维码扫码打不开 | 分享 URL 域名、`tenantCode`、路由 hash、二维码内容是否被截断 |
| 分享卡片图片不对 | 微信缓存、页面 meta 配置、首图是否可公网访问 |
| 保存图片无反应 | 微信相册权限、机型系统权限、海报是否已渲染完成 |
