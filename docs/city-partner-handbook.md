# 慢π城市合伙人说明书

更新时间：2026-06-11

## 1. 合作定位

慢π城市合伙人不是单纯加盟商，而是本地文化活动和课程运营者。平台提供 SaaS 后台、活动报名链路、会员工具、财务对账、活动审核和运营方法，合伙人负责本地资源、活动组织、用户服务和交付质量。

第一年重点跑通：

> 每个城市每月至少 2 场稳定活动，形成可复购的会员和课程报名模型。

## 2. 适合对象

| 类型 | 适合原因 |
| --- | --- |
| 文化空间 / 书院 | 有场地和本地文化用户基础 |
| 读书会 / 社群主理人 | 有社群信任和活动组织经验 |
| 书法 / 美育机构 | 适合做体验课、训练营和会员复购 |
| 家庭教育机构 | 用户付费意愿强，线下活动易转化 |
| 健康养生工作室 | 可做节气养生、身心减压、线下沙龙 |
| 创业技能社群 | 可做副业、私域、新媒体、企业内训 |

## 3. 平台提供什么

| 能力 | 说明 |
| --- | --- |
| 独立商家后台 | 合伙人可管理自己的活动、报名、订单、会员、公告、首页和员工 |
| 独立收款方式 | 支持绑定商家自己的微信/支付宝/线下收款配置，订单归属清晰 |
| 平台活动审核 | 平台统一把关活动合规、内容质量和发布状态 |
| 报名和签到工具 | 支持报名字段、报名审核、候补、签到核销和报名导出 |
| 票务和优惠工具 | 支持票种、优惠码、会员折扣、积分抵扣和优先报名 |
| 财务对账 | 支持订单、流水、退款、回调日志、对账差异和导出 |
| 员工权限 | 可设置运营、财务、签到人员，减少权限混乱 |
| 运营复盘 | 通过活动漏斗、报名、付款、签到和评价数据复盘活动效果 |

## 4. 合伙人负责什么

| 责任 | 要求 |
| --- | --- |
| 本地资源 | 提供场地、讲师、社群或渠道资源 |
| 活动组织 | 每月至少策划并执行 2 场活动 |
| 用户服务 | 负责报名咨询、现场接待、售后沟通和社群维护 |
| 内容合规 | 不发布算命、改运、破灾、预测财富/婚姻/疾病等内容 |
| 交付质量 | 确保活动真实、讲师可信、宣传与实际交付一致 |
| 财务配合 | 按平台要求维护收款资料、处理退款和对账异常 |

## 5. 首批活动方向

| 方向 | 活动示例 |
| --- | --- |
| 国学 / 东方哲学与传统文化 | 经典导读、节气文化、民俗文化、传统哲学沙龙 |
| 书法 / 美育 | 硬笔体验课、毛笔入门、亲子书法、成人美育 |
| 家庭教育 | 亲子沟通、学习力、青春期沟通、家长成长营 |
| 健康养生 | 节气养生、正念减压、体态改善、食疗入门 |
| 创业技能 | 私域运营、新媒体、短视频、副业启动、商业模式 |

第一阶段每个城市不建议五个方向同时做，先选择 1-2 个最有资源和付费基础的方向。

## 6. 开通流程

1. 资源沟通：确认城市、场地、讲师、社群、活动方向。
2. 合作确认：明确合伙人权益、责任、结算规则和合规边界。
3. 后台开通：平台创建商家租户、管理员账号、员工账号和权限。
4. 收款配置：商家绑定自己的收款方式，平台检查账户可用性。
5. 活动上线：发布 2-3 场样板活动，必要时提交平台审核。
6. 首月复盘：复盘报名转化、付款、签到、评价、退款和利润。

首月执行请按 [城市合伙人首月运营 SOP](city-partner-first-month-sop.md) 推进。

## 7. 权限说明

| 角色 | 权限 |
| --- | --- |
| 平台超级管理员 | 开通商家、配置权限、审核活动、监管全局订单和财务 |
| 商家管理员 | 管理本商家的活动、报名、订单、会员、收款方式和员工 |
| 商家运营 | 管理活动、公告、首页、通知、报名和用户标签 |
| 商家财务 | 管理订单、退款、对账和收款账户查看 |
| 商家签到 | 只做活动列表、报名查询和现场核销 |

## 8. 合规红线

禁止：

- 算命、改运、破灾
- 承诺消灾、转运、发财
- 预测婚姻、疾病、财富
- 以恐吓、焦虑、迷信话术推动成交
- 宣传与交付明显不一致

允许：

- 传统哲学解读
- 民俗和节气文化
- 人文地理和历史文化
- 国学经典导读
- 书法、美育、家庭教育、健康养生、创业技能等正常课程活动

## 9. 首月验收标准

| 指标 | 建议目标 |
| --- | --- |
| 样板活动数量 | 2-3 场 |
| 单场报名人数 | 10-30 人起步 |
| 到场率 | 70% 以上 |
| 退款率 | 5% 以下 |
| 满意评价 | 4.5 分以上 |
| 复购线索 | 每场沉淀可跟进用户 |
| 财务对账 | 订单、收款、退款、手续费归属清楚 |

## 10. 下一步

平台招商时优先寻找能提供场地、讲师或本地社群的合作方。开通后先做小规模样板活动，不急于大规模铺课程，用真实报名、复购和活动利润决定城市复制节奏。

正式合作前应先确认 [城市合伙人合作与结算规则](city-partner-cooperation-policy.md)，尤其是收入归属、退款责任、结算周期、违规处理和退出机制。
