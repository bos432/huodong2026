## 4. 演示前准备

```bash
npm run seed:qiwai-demo
npm run smoke:qiwai-demo
npm run report:qiwai-demo
```

命令说明：

| 命令 | 作用 |
| --- | --- |
| `npm run seed:qiwai-demo` | 创建三城合伙人、账号、收款主体和样板活动 |
| `npm run smoke:qiwai-demo` | 验证登录、隔离、报名、收款、签到、标签、复盘、退款、结算、审核 |
| `npm run report:qiwai-demo` | 生成三城样板经营报告 |

## 5. 推荐演示流程

1. 打开 H5 城市合伙人招商页，介绍合作定位和招商入口。
2. 用 `admin / Admin123456` 登录平台后台，查看商家管理、活动审核、全局订单和全局财务。
3. 在平台后台进入“运营监管 / H5全局装修”，查看首页分页、底部菜单、我的页和内页布局配置。
4. 在“商家/代理管理”列表点击某个城市合伙人的“预览H5”“复制链接”或“二维码”，也可以在“H5全局装修”选择商家后点击“打开H5预览”，确认链接自动携带 `tenantCode`。
5. 刷新 H5 用户端，检查首页活动分页、底部菜单、活动列表、公告中心、服务中心和活动详情的装修效果。
6. 点击 H5 首页左上角城市/商家名称，在底部弹窗中切换成都、杭州、苏州城市合伙人，确认活动、公告和装修内容随商家切换。
7. 进入 H5 活动页和我的页，使用顶部“当前城市”切换条再次切换城市合伙人，确认活动列表、分类、报名数据和页面装修同步刷新。
8. 进入 H5 公告中心、服务中心和城市合伙人招商页，使用顶部“当前城市”切换条切换城市合伙人，确认公告、客服联系方式、付款/退款/发票说明和招商联系信息同步刷新。
9. 进入活动详情、报名确认、报名详情、评价和登录页，确认顶部显示当前城市/商家归属提示；这些绑定活动、订单或报名的页面不提供切换按钮，避免误切商家导致订单边界混乱。
10. 用杭州商家管理员登录，查看商家概览、活动、报名、订单、会员和运营设置，并使用后台顶部“打开商家H5 / 复制商家H5 / 商家H5二维码”快速预览、转发或手机扫码访问本商家前台。
11. 在商家概览中查看经营驾驶舱，确认净收入、本月净收入、签到率、浏览转报名、经营健康和快捷处理入口。
12. 在活动管理页对某个活动点击“预览H5 / 复制链接 / 二维码 / 海报”，确认链接进入该商家的单场活动详情页，并下载一张带活动二维码的推广海报。
13. 用杭州财务账号登录，查看订单、财务对账和收款账户，并确认顶部 H5 快捷入口仍自动带杭州商家 `tenantCode`。
14. 用杭州签到账号登录，查看活动列表和报名查询。
15. 用杭州财务账号查看代理结算，演示结算单生成、提交审核、审核通过、打款登记、结算详情审计和导出。
16. 用杭州财务账号在订单管理中对某个订单点击备注按钮添加内部备注，确认保存后再次查看。
17. 打开 [慢π三城样板经营报告](qiwai-demo-sample-report.md)，展示三城 GMV、报名、签到率、结算金额、已打款金额和健康状态。

## 6. 当前已跑通能力

- 三城城市合伙人独立后台。
- 商家活动、报名、订单、会员、财务数据隔离。
- H5 公开端按 `tenantCode` 隔离活动和运营设置。
- H5 首页可直接切换城市合伙人，公开商家列表只展示启用的成都、杭州、苏州样板商家，不展示平台和 demo 租户。
- H5 活动页和我的页可直接切换城市合伙人，切换后自动刷新当前商家的主题、装修、活动和报名数据。
- H5 公告中心、服务中心和城市合伙人招商页可直接切换城市合伙人，切换后自动刷新公告、客服与运营说明等租户化内容。
- H5 活动详情、报名确认、报名详情、评价和登录页会显示当前商家/城市归属提示，交易链路资源页保持只读归属提示，不直接切换商家。
- 三城样板活动均已跑通报名、线下收款、签到、用户标签、活动复盘和财务可见。
- 商家概览已升级为经营驾驶舱，支持查看净收入、本月净收入、退款金额、签到率、浏览转报名、客单价、经营健康和快捷处理入口。
- 杭州样板已跑通部分退款，经营报告会体现退款金额和净收入变化。
- 城市合伙人结算已跑通结算单生成、提交审核、审核通过、打款登记、详情审计、导出和跨城隔离；本地未开启真实转账沙箱时自动走线下打款登记闭环。
- 平台超级管理员可跨商家监管。
- 平台商家列表和 H5 全局装修页可一键打开或复制对应城市合伙人的 H5 预览链接，减少手动拼 `tenantCode`。
- 后台顶部栏已提供常驻 H5 快捷入口，平台账号打开平台 H5，商家管理员、运营、财务和签到账号会自动打开或复制当前商家的 H5 链接。
- 后台顶部栏和平台商家列表已支持 H5 二维码弹窗，可复制链接或下载二维码图片，方便现场活动和招商测试扫码。
- 订单管理已支持财务/运营人员添加内部备注，备注仅后台可见。
- 活动管理页已支持单场活动 H5 详情链接、二维码和推广海报，平台视角自动使用活动所属商家的 tenantCode，商家视角自动使用当前商家编码。
- 平台超级管理员可配置 H5 全局装修，包含首页活动分页、底部菜单、我的页、活动列表、公告中心、服务中心、城市合伙人招商页、活动详情、报名确认、报名详情、评价和登录页内页布局。
