# 七维文化城市合伙人合作与结算规则

更新时间：2026-06-12

## 1. 合作原则

城市合伙人是七维文化在本地的活动和课程运营主体，不是单纯代理销售。平台提供品牌、SaaS 后台、活动报名链路、审核监管、会员和财务工具；城市合伙人负责本地资源、活动交付、用户服务和基础经营结果。

第一阶段目标：

> 每个城市先跑通 2-3 场样板活动，形成可复用的活动 SOP、讲师资源、用户社群和财务复盘。

## 2. 合作对象与准入

| 准入项 | 要求 |
| --- | --- |
| 城市资源 | 有场地、社群、讲师、机构渠道或本地内容资源之一 |
| 运营能力 | 能独立组织活动、维护社群、处理报名咨询和现场执行 |
| 合规意识 | 接受平台内容审核，不做算命、改运、破灾、预测类宣传 |
| 服务意识 | 能承担用户咨询、退款沟通、发票登记和售后协调 |
| 财务配合 | 能提供真实收款资料，按平台要求处理对账和结算 |

不建议准入：

- 只有招商意愿但没有本地资源的人。
- 无法承担用户服务和退款沟通的人。
- 以封建迷信、焦虑营销、夸大承诺作为主要成交方式的人。
- 无法接受平台审核和运营数据复盘的人。

## 3. 合作版本建议

| 版本 | 适用对象 | 建议费用 | 权益 |
| --- | --- | ---: | --- |
| 试运营合伙人 | 有资源但未验证活动模型 | 0-3000 元保证金或服务费 | 开通后台、发布 2-3 场样板活动、平台协助复盘 |
| 城市合伙人 | 能稳定每月办活动 | 1-3 万元/年 | 独立商家后台、品牌授权、活动模型、运营培训、优先讲师资源 |
| 城市核心合伙人 | 有团队和多场地资源 | 3-5 万元/年 | 城市重点扶持、联合招商、平台活动资源优先、月度经营复盘 |

说明：

- 第一阶段建议先采用“试运营 -> 正式合作”的路径，不急于收高额加盟费。
- 试运营期以真实报名、收款、签到、退款率和复购线索作为是否升级的依据。
- 最终价格可按城市资源、场地投入、讲师资源和平台支持深度调整。

## 4. 收入归属与分成

### 4.1 活动报名收入

| 场景 | 建议规则 |
| --- | --- |
| 合伙人自有讲师、自有场地 | 扣除支付手续费和退款后，合伙人获得 70%-90%，平台获得 10%-30% 技术和品牌服务费 |
| 平台提供讲师，合伙人提供场地和用户 | 讲师、合伙人、平台按活动前约定比例分成，建议先按 40% / 40% / 20% |
| 平台统一策划和投放，合伙人执行 | 扣除投放和讲师成本后按净利润分成，比例单独约定 |
| 公益或低价引流活动 | 可不分成，以用户沉淀、会员转化和后续课程为目标 |

### 4.2 会员年卡收入

| 来源 | 建议规则 |
| --- | --- |
| 合伙人本地转化会员 | 合伙人获得首年 30%-50% 推荐服务费 |
| 平台公域或总部转化会员 | 平台归属，合伙人承接本地活动时可按活动分成 |
| 会员续费 | 续费归属按用户主要活跃城市、服务成本和平台规则核算 |

### 4.3 训练营和高客单课程

| 场景 | 建议规则 |
| --- | --- |
| 本地训练营 | 按讲师成本、场地成本、平台服务费后核算净利润 |
| 企业内训 | 谁获取客户谁获得销售服务费，交付方按项目毛利分成 |
| 私董会/陪跑服务 | 单独立项，明确交付边界、退款条款和负责人 |

## 5. 结算规则

| 项目 | 规则 |
| --- | --- |
| 结算周期 | 默认按月结算；活动密集城市可按半月结算 |
| 结算口径 | 实收金额 - 已完成退款 - 支付手续费 - 明确成本 |
| 结算资料 | 订单明细、退款明细、签到记录、活动复盘、异常说明 |
| 打款方式 | 优先打到合伙人已备案账户；线下打款需上传凭证 |
| 异常处理 | 存在未处理退款、投诉、对账差异时可暂缓相关活动结算 |

平台后台当前支持：

- 商家独立收款账户。
- 财务对账和导出。
- 代理/渠道结算单。
- 打款凭证附件。
- 结算审计记录。

## 6. 退款与售后责任

| 情况 | 责任建议 |
| --- | --- |
| 用户未参加且符合退款规则 | 按活动页退款说明处理 |
| 活动取消或延期 | 由组织方优先承担退款和沟通责任 |
| 宣传与交付不一致 | 由发布活动的商家/合伙人承担主要责任，平台可介入处理 |
| 讲师临时变更 | 需提前通知用户，必要时允许退款 |
| 不合规内容导致投诉 | 平台可下架活动、暂停商家权限并冻结未结算款项 |

退款规则必须在活动详情页提前说明，不允许现场临时更改。

## 7. 内容审核与违规处理

禁止内容：

- 算命、改运、破灾。
- 承诺发财、转运、消灾。
- 预测婚姻、疾病、财富。
- 恐吓式营销和制造焦虑成交。
- 夸大疗效、虚假认证、虚假讲师履历。

处理机制：

1. 首次轻微违规：驳回活动，要求修改文案。
2. 重复违规：暂停活动发布权限。
3. 严重违规：下架活动、暂停商家后台、暂缓结算。
4. 产生投诉或监管风险：平台保留终止合作和追责权利。

## 8. 首月考核

| 指标 | 达标线 |
| --- | --- |
| 样板活动 | 至少 2 场 |
| 报名人数 | 单场 10-30 人起步 |
| 收款闭环 | 至少 1 场完成报名、收款、签到、复盘 |
| 到场率 | 70% 以上 |
| 退款率 | 5% 以下 |
| 用户标签 | 每场沉淀高意向、活跃用户、复购线索 |
| 复盘报告 | 每场活动结束后 48 小时内完成 |

未达标时，不建议扩大投放或开新城市；应先复盘活动主题、价格、渠道、讲师和客服话术。

## 9. 退出与暂停

平台可暂停合作：

- 连续 2 个月无有效活动。
- 多次活动投诉且未整改。
- 财务对账长期不配合。
- 发布不合规内容。
- 私下绕开平台收款且损害用户权益。

合伙人可申请退出：

- 完成已报名活动交付。
- 处理完退款、投诉和未结算事项。
- 交接用户服务和财务记录。
