# 七维文化城市合伙人首月验收清单

更新时间：2026-06-12

## 1. 使用场景

本清单用于七维文化第一阶段城市样板验收，重点确认：

- 城市合伙人后台是否真正可独立运营。
- 首批活动是否跑通报名、收款、签到、复盘和财务。
- 平台是否能监管活动审核、商家权限、财务和结算。
- 首月数据是否支持继续投入、调整方向或暂停合作。

适用对象：

| 对象 | 用途 |
| --- | --- |
| 平台负责人 | 判断城市样板是否达标 |
| 城市合伙人 | 对照完成首月经营动作 |
| 商家运营 | 检查活动、报名、会员和复盘 |
| 商家财务 | 检查订单、退款、对账、结算和打款 |
| 商家签到 | 检查现场核销和报名查询 |

## 5. 首月经营数据验收

| 指标 | 达标线 | 数据来源 | 结果 |
| --- | ---: | --- | --- |
| 样板活动数 | 每城 2 场以上 | 经营报告、后台活动列表 | 待勾选 |
| 单场报名 | 建议 10 人以上 | 报名管理、活动复盘 | 待勾选 |
| 已收款订单 | 至少 1 场跑通 | 订单管理、财务看板 | 待勾选 |

| 复购线索 | 每场至少 3 个 | 用户标签、运营记录 | 待勾选 |
| 下月排期 | 至少 2 场 | 活动计划、商家沟通 | 待勾选 |
| 结算清晰 | 订单、退款、结算、打款可追溯 | 结算详情、导出 | 待勾选 |

## 6. 合规验收

活动、公告、详情页、客服话术均不得出现：

- 算命、改运、破灾。
- 承诺发财、转运、消灾。
- 预测婚姻、疾病、财富。
- 夸大疗效、虚假讲师资质。
- 恐吓式营销或制造焦虑成交。

允许表达：

- 东方哲学与传统文化。
- 民俗文化、节气文化、国学经典解读。
- 书法、美育、家庭教育、健康养生、创业技能等正常课程活动。

## 7. 验收结论

| 结论 | 判断标准 | 后续动作 |
| --- | --- | --- |
| 通过 | 系统闭环通过，经营指标基本达标，无重大合规风险 | 进入月度运营，安排下月活动 |
| 有条件通过 | 系统闭环通过，但报名、签到、复购或结算有短板 | 限期优化主题、渠道、客服或财务流程 |
| 暂缓 | 活动未落地、收款未跑通、投诉或合规风险明显 | 暂停扩大投放，重新评估合伙人 |
| 不通过 | 数据串租户、财务不可追溯、严重违规或无法交付 | 暂停合作或重新开通样板城市 |

## 8. 归档要求

每个城市首月验收后至少归档：

- 城市合伙人资料和后台账号清单。
- 首批活动链接和活动截图。
- 报名、订单、签到、退款、结算导出。
- 活动复盘记录。
- 用户标签和复购线索记录。
- 合规审核记录和整改记录。
- 是否继续投入的结论。
